# HR Analytics: Employee Attrition & Hiring Predictive Model

This repository contains an end-to-end Machine Learning pipeline designed to assist HR professionals in predicting employee attrition and streamlining the recruitment process. The project achieves an **85.59% F1-Score** and an **ROC-AUC of 93.09%**.

##  Project Overview
The goal of this project is to identify key drivers of employee turnover and provide a decision-support system for hiring. By automating the initial screening, the model saves approximately **196 hours per recruitment cycle**.

### Key Features:
- **Exploratory Data Analysis (EDA):** Insights into job satisfaction, monthly income, and work-life balance.
- **Machine Learning:** Utilizes Random Forest with feature importance tracking.
- **Bias Audit:** Includes a DEI (Diversity, Equity, and Inclusion) equity analysis to ensure fairness.
- **Business Impact:** Quantitative analysis of time and cost savings.

##  File Structure
- `2.ipynb`: The main Jupyter Notebook containing data preprocessing, model training, and evaluation.
- `2.csv`: The dataset containing employee attributes (Age, Department, MonthlyIncome, etc.).
- `*.png`: Visualizations including the EDA Dashboard, Model Comparison, and Bias Audit.

##  Performance Metrics
- **F1 Score:** 85.59%
- **ROC-AUC:** 93.09%
- **Business Savings:** Est. $4,900 per recruitment cycle.

##  Ethical Notice
This tool is a decision-support system. All AI recommendations must be reviewed by a qualified HR professional before any candidate communication or employment decisions are made.
